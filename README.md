# Plagiarism Detector Using String Matching Algorithms

## Project Overview

This project is a DSA-focused plagiarism detection system designed to identify exact and near-duplicate content using multiple string matching algorithms and similarity techniques.

## Problem Statement

Manual plagiarism checking is slow and inefficient. This project automates plagiarism detection by comparing documents using classical algorithms and similarity metrics.

## Features

* Text preprocessing
* File handling
* Exact matching using KMP
* Rabin-Karp implementation
* Similarity score calculation
* Report generation
* Graph visualization
* GitHub-ready project structure

## Algorithms Used

### KMP Algorithm

Used for exact pattern matching with efficient preprocessing.

### Rabin-Karp Algorithm

Uses hashing for faster substring matching.

### TF-IDF Similarity

Measures semantic similarity between documents.
